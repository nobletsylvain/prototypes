# VARIÉTÉS — l'échelle de qualité (catalogue complet)

**Statut :** proposition de système, compagnon de `CADRE.md` (§1, curseur
Satisfaction & note R10). **Aucun chiffre figé** — placeholders à régler.
Visuel interactif (échelles + démo déception) : voir l'artefact publié.

Principe fondateur (Sylvain) : **une variété = un cran de QUALITÉ, pas un goût.**
Les noms réels servent de **repères de qualité/marque**, pas de préférence
gustative. La qualité tient au **mode de production** (weed/hash), à la
**pureté / place dans la chaîne d'import** (coke, héro, kéta, speed, meth), à la
**dose** (ecsta, LSD), à la **concentration** (GHB), ou à l'**authenticité**
(3-MMC, tucibi, LSD).

**Produits couverts :** hash · weed · coke · héro · crack · ecsta · MDMA ·
3-MMC · kétamine · speed · méthamphétamine · tucibi/2C-B · LSD · GHB · champis.

---

## 1. Une variété = 4 stats

| Stat | Rôle |
|---|---|
| **Qualité de base** (grade) | alimente la **Satisfaction** (CADRE §1) |
| **Prix de marché** (~fixe) | monte avec le tier ; le joueur ne le fixe pas librement |
| **Coût de revient** | monte avec le tier → détermine la marge |
| **Segment de clients** | qui achète ce tier, et sa **sensibilité à la déception** |

**Segment → sensibilité** : *bas de gamme* = gros réservoir, tolérant ;
*connaisseurs* = petit réservoir, paient cher, **hyper-sensibles**.

## 17. La mécanique de déception (le cœur)

Deux valeurs séparées à chaque vente : ce que tu **annonces** (pub/menu/marque)
et ce que tu **livres** (tier réel, éventuellement charclé). Résolution **100 %
déterministe** (R4) :

- **Livré ≥ annoncé** → **satisfait/ravi** → réservoir se **remplit**.
- **Livré < annoncé** → **la vente se fait**, mais **déception = (annoncé −
  livré) × sensibilité(segment annoncé)** → réservoir s'**érode**.

Sur-promettre + sous-livrer = **cash maintenant, clients perdus demain**.
Tromper le bas de gamme est indolore ; tromper les connaisseurs vide le réservoir
haut de gamme d'un coup. **Déception native** sur les produits « à arnaque »
(coke écaille, 3-MMC/3-CMC, tucibi, LSD/NBOMe).

## 18. Ce que ça relie dans le CADRE

- **Charcler (note R10)** = **descendre d'un ou plusieurs tiers**. Iconique sur
  coke, héro, speed, 3-MMC, tucibi.
- **Grimper l'échelle** — quatre ascenseurs selon le produit :
  - **production** : weed, hash, champis (culture) ;
  - **sourcing d'import** : coke, héro, kéta, meth ;
  - **cuisson/transformation** : crack (depuis coke) ;
  - **synthèse / presse / mélange** : ecsta, MDMA, 3-MMC, speed, LSD, GHB,
    tucibi (labo, presse, ou recette).
  Toujours : **monter les tiers en gardant la marge** (intégration).
- **Pipelines** : weed → hash ; coke → crack ; (kéta + MDMA → tucibi). Un
  produit **alimente** un autre et **ouvre un marché**.
- **Substitution** : **partielle** intra-produit ; **nulle** entre produits
  (même ecsta ↔ MDMA).
- **Segments & demande** : dépendants (héro, crack, meth) = réservoirs stables ;
  festifs (ecsta, MDMA, kéta, LSD, champis) = **pics week-end** ; chemsex (3-MMC,
  GHB, meth) = urbain nocturne ; masse continue (hash, weed, coke, speed).

## 19. Ouvert / à définir

- Grey-market (poppers, protoxyde/proto, lean/codéine) : hors périmètre pour
  l'instant (statut légal ambigu).
- **Degré exact de substitution** intra-produit.
- **Rendu du charcler** en tiers (perte fractionnaire vs saut d'un cran).
- **Pipelines** (rendements weed→hash, coke→crack, kéta+MDMA→tucibi, synthèses).
